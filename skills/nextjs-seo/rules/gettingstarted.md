## Prerequisites
- **App Router**
- **Next.js 13.0.0 or higher**

## Next.js Native SEO
This skill helps you implement SEO metadata, sitemaps, robots.txt, and JSON-LD structured data using standard Next.js built-in features. No external packages are required!
